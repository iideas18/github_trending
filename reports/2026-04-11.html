A single CLAUDE.md file to improve Claude Code behavior, derived from Andrej Karpathy's observations.

This project distills Andrej Karpathy's observations about LLM coding pitfalls into a single CLAUDE.md configuration file that dramatically improves Claude Code's behavior. Karpathy noted that AI models "make wrong assumptions on your behalf," "overcomplicate code and APIs," and "change/remove comments and code they don't understand as side effects." This project addresses all three issues through four clear principles.

The four principles are: Think Before Coding (surface assumptions and tradeoffs instead of guessing), Simplicity First (minimum code that solves the problem with nothing speculative), Surgical Changes (touch only what you must, clean up only your own mess), and Goal-Driven Execution (leverage tests-first and verifiable success criteria). Each principle directly maps to specific LLM failure modes.

The project is trending because it offers an immediately actionable improvement for anyone using Claude Code. Drop a single file into your project, and your AI coding assistant becomes more thoughtful, less bloated, and more respectful of existing code — addressing the exact frustrations that experienced developers have with AI coding tools.

Kronos: A Foundation Model for the Language of Financial Markets

Kronos is the first open-source foundation model specifically designed for financial candlestick (K-line) data, trained on data from over 45 global exchanges. Accepted by AAAI 2026, it represents a novel approach to financial time-series modeling by treating market data as a "language" that can be learned through pre-training, similar to how LLMs learn natural language.

The model uses a two-stage framework: a specialized tokenizer first quantizes continuous, multi-dimensional K-line data (OHLCV — Open, High, Low, Close, Volume) into hierarchical discrete tokens, and then a large autoregressive Transformer is pre-trained on these tokens. This enables Kronos to serve as a unified model for diverse quantitative tasks including prediction, pattern recognition, and anomaly detection across markets.

Kronos is trending in the intersection of AI and finance — it provides researchers and quant developers with a pre-trained foundation model they can fine-tune for their own financial tasks, rather than training from scratch. The availability of fine-tuning scripts and a live demo makes it accessible for experimentation and adaptation.
