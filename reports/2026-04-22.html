Code search MCP for Claude Code. Make entire codebase the context for any coding agent.

claude-context is a Model Context Protocol (MCP) server from Zilliz that gives AI coding agents like Claude Code, Cursor, and VSCode extensions the ability to perform fast, semantic search across an entire codebase. Instead of forcing agents to scan files one-by-one or rely on naive grep, it indexes a repository into a vector store and exposes a standardized MCP interface so the agent can retrieve just the relevant chunks for any query.

The project combines a TypeScript-based MCP server with an embedding-backed code index, and uses techniques like Merkle-tree change detection to keep the index cheap and incremental as files evolve. It is designed to plug into multiple agent runtimes and editors without requiring users to re-upload large codebases for every session.

With MCP becoming the de facto way to wire tools into coding agents, claude-context is trending because it targets a real bottleneck: context windows are finite, so teams need retrieval that is code-aware rather than document-generic. It will appeal to developers who want more reliable, whole-repo reasoning from their AI pair programmer without burning tokens on irrelevant files.

A flexible enhancer for YouTube on iOS

YTLite is a feature-rich tweak for the YouTube iOS app aimed at jailbroken devices and sideloaders. Built in Logos (the Theos hooking language for iOS tweaks), it patches the official YouTube binary to add quality-of-life improvements the stock app lacks: ad blocking, background playback, picture-in-picture on any tier, custom themes, and fine-grained UI toggles.

It also bundles integrations like a video/audio downloader and SponsorBlock support to auto-skip sponsor segments, intros, and self-promotion. Because it is 'flexible,' almost every tweak can be toggled from an in-app settings panel so users can opt in or out per feature instead of inheriting one maintainer's preferences.

YTLite is trending now thanks to active releases that keep up with YouTube's frequent client updates, which often break older tweaks. It will appeal to iOS power users in the jailbreak/sideload community looking for a maintained, all-in-one YouTube enhancer rather than juggling several smaller tweaks.

"RAG-Anything: All-in-One RAG Framework"

RAG-Anything, from the HKU Data Science Lab, is an all-in-one retrieval-augmented generation framework that aims to handle essentially any input modality—plain text, PDFs with complex layouts, tables, equations, images, and more—through a single unified pipeline. It pairs document parsing and multi-modal embedding with a retrieval + generation stack so developers don't have to stitch together separate toolchains for each file type.

The framework ships with configurable parsers, a modality-aware indexer, and a query engine that can route questions across text, visual, and structured content. It is designed to interoperate with popular LLM backends and vector stores, letting teams plug in their own models while still benefiting from the end-to-end orchestration.

Multi-modal RAG is one of the hottest areas in applied AI right now, and most production stacks struggle with mixed-content documents. RAG-Anything is trending because it packages recent research advances from the HKUDS group into a practical, Python-first framework, making it attractive to researchers and engineers building knowledge assistants over real-world document collections.
