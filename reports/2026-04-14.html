Create and share 3D architectural projects.

Pascal Editor is an open-source 3D building editor built with React Three Fiber and WebGPU, designed for creating and sharing architectural projects directly in the browser. It provides a full-featured scene editor where users can draw walls, place doors and windows, create floor slabs, define zones, and navigate through multi-level building structures—all rendered in real-time 3D.

The project is architected as a Turborepo monorepo with clean separation of concerns: a core package handles node schemas, state management via Zustand, and geometry generation systems; a viewer package manages 3D rendering with React Three Fiber; and the editor app adds interactive tools and UI. Notable technical features include WebGPU rendering, undo/redo support via Zundo, CSG boolean operations for wall cutouts, and a dirty-node system that efficiently updates only changed geometry each frame.

This project is likely trending due to the growing interest in browser-based 3D design tools and the maturity of the WebGPU ecosystem. Architects, interior designers, game developers prototyping level layouts, and React/Three.js developers looking for a well-structured reference implementation would all find this project valuable.

Original Apollo 11 Guidance Computer (AGC) source code for the command and lunar modules.

This repository contains the original source code for the Apollo 11 Guidance Computer (AGC), which navigated humanity’s first Moon landing in 1969. It includes the complete programs for both the Command Module (Comanche055) and the Lunar Module (Luminary099), digitized from hardcopy printouts held at the MIT Museum in collaboration with the Virtual AGC project.

The code is written in AGC Assembly language and represents one of the most historically significant software projects ever created. It was developed at the MIT Instrumentation Laboratory under the leadership of Margaret Hamilton, whose team pioneered many concepts in software engineering. The repository faithfully preserves the original assembly listings, including the famous programmer comments and annotations from 1969.

The Apollo 11 source code periodically resurfaces on trending lists as new audiences discover this remarkable piece of computing history. It appeals to software engineers curious about the origins of mission-critical systems, space enthusiasts, historians of technology, and educators teaching computer science. With over 57,000 stars, it remains one of the most celebrated open-source repositories on GitHub.
